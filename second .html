    Whta is Nesterted List

    • A nested list in HTML means placing one list inside another list.
    • it helps create sub-items or a hierarchical structure—like main topics with subtopics.
    • The inner list can be ordered (ol) or unordered (ul), regardless of the outer list type.
    • The nested ul or ol must be inside a li tag, not directly inside ul or ol.

    What is Rows and Column?

    • Row:- A row is a horizontal line of cells (left to right).
    • Column:-A column is a vertical set of cells (top to bottom) in the table.

    What is Table?

    A Table is an arrangement of data made up of:
    • Row:- A row is a horizontal line of cells (left to right).
    • Column:-A column is a vertical set of cells (top to bottom) in the table.
    • Cell:- the individual boxes where rows and columns intersect

    What is Table Tag

    • The table tag in HTML is used to create tables—a structured way to present data in rows and columns.
    • A table is made up of several child tags that define rows, headers, and data cells.
    • attributes of the table tag:-
      1. Border:- Sets border thickness in pixel.
      2. Cellpadding:- add space inside each cell
      3. Cellspacing:- add space between cells
      4. Width:- sets table width
      5. Align:- align table left, right, center

    What is the child tags of the table tag?

    • thead tag:- Groups header rows (optional).
    • tbody tag:- Groups main data rows.
    • tfoot tag:- Groups footer rows (totals, notes).
    • tr tag:- Table Row – creates a horizontal row of cells.
    • d tag:- Table Data Cell – Insert Data in the cell.
